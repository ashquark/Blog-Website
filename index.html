Blog Entry: My First Blog!

Date: 15-06-2024, Saturday

Hi, I am Ashutosh Sharma.

I am starting this blog as a personal project and as a way to document my foray into web development. I hope to commit to this blog once a day at least. Today is a routine Saturday, I got some time for myself free of the weekday chore of my job. I wanted to start this project for quite some time now, finally as I write my pilot entry I feel a sense of productivity which I so crave.

This first entry is made using just an html file and nothing else, using only basic html. I hope to expand the technology used in this project gradually. I have thought of starting with MERN stack at first, but I will be flexible in using anything new that I learn. I know that this project will not be too complex or some state of the art project or even a novel idea, blogs have been used to their full extent already. The idea behind this project is to just start with the basics and then diverge into more complex, even novel, endeavors. I want to add something new to the project with each blog entry. (Spoiler Alert!) For my next entry I will use CSS.

The content I have decided will mostly be based on my journey deeper into programming, web development and software engineering but will not be limited to that. I might include the some excerpts about my day, log my daily efforts and even about myself. There is a possibility of a diary entry being squeezed into the mix. I might rediscover my passion for writing through this blog. So this blog satifies multiple requirements and hence I chose to start with it. For now the entries are safely stored (I hope!) on my PC, but before long I will host it. That's it for now. Logging out.

Fin.

Ash
